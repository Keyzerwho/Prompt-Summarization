 You are given a number N. Write a program to determine whether it is one digit number, 2 digit number, 3 digit number or more than 3 digit number.   Example 1: Input: N = 2 Output: 1 Explanation: 1 is a one digit number. Example 2: Input: N = 3 Output: 2 Explanation: 3 is a two digit number. Example 3: Input: N = 4 Output: 2 Explanation: 4 is a three digit number
-----Input:----- - First line will contain the number $N$, -----Output:----- Print "1" if N is a 1 digit number. Print "2" if N is a 2 digit number. Print "3" if N is a 3 digit number. Print "More than 3 digits" if N has more than 3 digits. -----Constraints----- - $0 \leq N \leq 1000000$ -----Sample Input:----- 9 -----Sample Output:----- 1
