 Given an array V of integers, find number of ways Tejas can serve the Clash Team. 
-----Input----- - The first line of the input contains an integer T denoting the number of test cases . The description of T testcases follow. - The first line of each test case contains two space seperated integers V W . -----Output----- For each test case, output the number of ways Tejas can serve the Clash Team. -----Constraints----- - 1 ≤ T ≤ 100 - 1 ≤ V ≤ 1000 - 1 ≤ W ≤ 1000 -----Example----- Input: 1 3 3  Output: 4 -----Explanation----- In the above example the 4 ways are: AAA AAB ABB BBB
