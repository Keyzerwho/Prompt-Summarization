 You are given $k$ number of chocolates, each containing some number of chocolates. Tell whether dhiraj will be able to eat $k$ number of chocolates or not. 
-----Input:----- - First line will contain $T$, number of testcases. Then the testcases follow.  - $k$, representing the number of chocolates dhiraj has to eat. - the third line contains $N$ representing the no. of boxes of chocolates. - fourth line contains list of $a[]$ size $N$ specifying the number of chocolates in each Box. -----Output:----- - For each testcase, output in a single line answer $0$ or $1$. - $0$ if dhiraj cant eat $k$ chocolates from given combination and $1$ if he can eat $k$ chocolates from given combination. -----Constraints----- - $1 \leq T \leq 100$ - $1 \leq K \leq 10^7$ - $1 \leq  N \leq 800$ - $1 \leq a[i] \leq 10^3$ -----Sample Input:----- 2 20 5 8 7 2 10 5 11 4 6 8 2 10 -----Sample Output:----- 1 0
