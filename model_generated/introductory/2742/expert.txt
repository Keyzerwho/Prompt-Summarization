 Given a string and an array of index numbers, return the characters of the string rearranged to be in the order specified by the accompanying array. Ex: scramble('abcd', [0,3,1,2]) -> 'acdb'  The string that you will be returning back will have: 'a' at index 0, 'b' at index 3, 'c' at index 1, 'd' at index 2, because the order of those characters maps to their corisponding numbers in the index array.  In other words, put the first character in the string
