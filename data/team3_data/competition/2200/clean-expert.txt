You need to compute $$ (a_1 + a_2) \oplus (a_1 + a_3) \oplus \ldots \oplus (a_1 + a_n) \\ \oplus (a_2 + a_3) \oplus \ldots \oplus (a_2 + a_n) \\ \ldots \\ \oplus (a_{n-1} + a_n) \\ $$ Here $x \oplus y$ is a bitwise XOR operation. Input: The first line contains a single integer $n$ ($2 \leq n \leq 400\,000$) — the number of integers in the array. The second line contains integers $a_1, a_2, \ldots, a_n$ ($1 \leq a_i \leq 10^7$). Output: Print a single integer — xor of all pairwise sums of integers in the given array. Examples: Input 2 1 2 Output 3 Input 3 1 2 3 Output 2
