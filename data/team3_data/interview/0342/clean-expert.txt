Given an 2 dimentional array count number of structures in it. Structures are represented by 'X's and empty slots are represented with "."s. Structures can only be horizontal or vertical. There are no adjacent structures. Example: X..X ...X ...X In the above 2 dimentional array there are 2 structures. Invalid Example: ...X XXXX ...X This is an invalid 2 dimentional array that you will not receive.
