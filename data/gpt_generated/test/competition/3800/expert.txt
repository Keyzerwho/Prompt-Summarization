 Given a string of decimal digits, find the number of rectangles in a matrix such that the sum of all cells in the rectangle equals a. 
-----Input----- The first line contains integer a (0 ≤ a ≤ 10^9), the second line contains a string of decimal integers s (1 ≤ |s| ≤ 4000). -----Output----- Print a single integer — the answer to a problem. Please, do not write the %lld specifier to read or write 64-bit integers in С++. It is preferred to use the cin, cout streams or the %I64d specifier. -----Examples----- Input 10 12345 Output 6 Input 16 439873893693495623498263984765 Output 40
