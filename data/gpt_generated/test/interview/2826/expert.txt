 Given values $q$ and $n$, focal points $f_1$ and $f_2$ and distance $D$, we ask you to determine the number of points $p \in F_ q^n$ that lie on this Hamming ellipse. 
-----Input----- The first line contains three integers $q$ ($2 \le q \le 10$), $n$ ($1 \le n \le 100$) and $D$ ($1 \le D \le 2 n$). The second and third lines specify the two focal points $f_1$ and $f_2$, each formatted as a string of length $n$ using digits $\{ 0, 1 \ldots q - 1\} $. -----Output----- Output one line containing a single integer, denoting the number of points on the ellipse. The input is chosen such that the answer is less than $2^{63}$. -----Examples----- Sample Input 1: 3 5 9 01201 21210 Sample Output 1: 24 Sample Input 2: 4 6 5 123031 231222 Sample Output 2: 0
