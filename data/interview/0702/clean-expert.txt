Given numbers $m$, $Tc$, $Th$. If the difference between $Tc$ and $Th$ is not divisible by $3$, return "Yes". Else if the differnce divided by $3$ is more than $m$ return "Yes". Else return "No". -----Input:----- - First line will contain $T$, number of testcases. Then the testcases follow. - Each testcase contains of a single line of input, three integers $m, Tc,Th$. -----Output:----- For each testcase, output based on the rules. -----Constraints----- - $1 \leq T \leq 100000$ - $1 \leq m,Tc,Th \leq 10^9$ - $Tc \leq Th $ -----Sample Input:----- 2 4 5 10 2 2 5 -----Sample Output:----- Yes No
