You are given an array of size n and a limiting number k. You can select only those numbers from array whose sum is not greater than k.Find the maximum number from the array which can be selected. Input Format The first line contains two integers, n and k , the number of elements in array and a limiting number. The next line contains n space-separated integers array[i] Constraints 1<= n <= 105 1<= k <= 109 1<= array[i] <= 109 A number can't be repeated multiple times. Output Format An integer that denotes the maximum count of array numbers whose sum can't be greater than k. Sample Input 7 50 1 12 5 111 200 1000 10 Sample Output 4
