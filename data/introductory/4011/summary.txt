## Your task:
You need to write a function that returns the maximum of free urinals as an integer according to the unwritten rule that you leave at least one urinal free between you and the next person peeing.

### Input
A String containing 1s and 0s (1 <= Length <= 20)  
A one stands for a taken urinal and a zero for a free one. 

### Examples
`10001` returns 1
`1001` returns 0
`00000` returns 3
`0000` returns 2
`01000` returns 1
