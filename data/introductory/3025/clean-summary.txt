Write a function that gets a sequence and value and returns `true` if the variable exists in a multidimentional sequence, otherwise false. Example: ``` locate(['a','b',['c','d',['e']]],'e'); // should return true locate(['a','b',['c','d',['e']]],'a'); // should return true locate(['a','b',['c','d',['e']]],'f'); // should return false ```
