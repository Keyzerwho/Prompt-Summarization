Write a function that takes an arbitrary number of strings and interlaces them. If the strings are different lengths the function should interlace them until each string runs out, continuing to add characters from the remaining strings. If only one argument is passed return only that string. If no arguments are passed return an empty string. Example 1: Input: `combineStrings('abc', '123')` Output: `'a1b2c3'` Example 2: Input: `combineStrings('abcd', '123')` Output: `'a1b2c3d'` Example 3: Input: `combineStrings('abc', '123', '£$%')` Output: `'a1£b2$c3%'`
