# Task You will be given an array of random integers and a number n. You have to extract n smallest integers out of it. The original order need to be preserved. # Examples performant_smallest([1, 2, 3, 4, 5], 3) == [1, 2, 3] performant_smallest([5, 4, 3, 2, 1], 3) == [3, 2, 1] performant_smallest([1, 2, 3, 4, 1], 3) == [1, 2, 1] performant_smallest([2, 1, 3, 2, 3], 3) == [2, 1, 2]
