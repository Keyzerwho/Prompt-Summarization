 You are given an integer sequence. Out of those 10^N sequences, how many contain an X,Y,Z-Haiku 
-----Input----- The input is given from Standard Input in the following format: N X Y Z -----Output----- Print the number of the sequences that contain an X,Y,Z-Haiku, modulo 10^9+7. -----Sample Input----- 3 5 7 5 -----Sample Output----- 1 Here, the only sequence that contains a 5,7,5-Haiku is [5, 7, 5].
