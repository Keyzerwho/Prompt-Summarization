 You are given a positive integer $n$, and $a_1, a_2, \ldots, a_n$ are positive integers. You need to find the number of seconds it would take to make Alice happy.
-----Input----- The first line contains a single integer $n$ ($1 \le n \le 10^5$) — the number of chocolate boxes. The second line contains $n$ integers $a_1, a_2, \ldots, a_n$ ($0 \le a_i \le 1$) — the number of chocolate pieces in the $i$-th box. It is guaranteed that at least one of $a_1, a_2, \ldots, a_n$ is positive. -----Output----- If there is no way for Charlie to make Alice happy, print $-1$. Otherwise, print a single integer $x$ — the minimum number of seconds for Charlie to help Bob make Alice happy. -----Examples----- Input 3 1 0 1 Output 2 Input 1 1 Output -1
