 
-----Input----- Th first line contains one integer $t$ ($1 \le t \le 1000$) — the number of test cases. The only line of each test case contains one integer $n$ ($1 \le n \le 1000$) — the number of windows in the building. -----Output----- For each test case, if a building with the new layout and the given number of windows just can't exist, print $-1$. Otherwise, print three non-negative integers — the possible number of three-room, five-room, and seven-room apartments. If there are multiple answers, print any of them. -----Example----- Input 4 30 67 4 14 Output 2 2 2 7 5 3 -1 0 0 2
