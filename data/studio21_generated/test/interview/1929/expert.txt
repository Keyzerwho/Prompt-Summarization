 You are given n prisoners. Choose c prisoners such that chosen c prisoners form a contiguous segment. Also, choose c prisoners such that no prisoner's crime level is greater than t. 
-----Input----- The first line of input will contain three space separated integers n (1 ≤ n ≤ 2·10^5), t (0 ≤ t ≤ 10^9) and c (1 ≤ c ≤ n). The next line will contain n space separated integers, the i^{th} integer is the severity i^{th} prisoner's crime. The value of crime severities will be non-negative and will not exceed 10^9.  -----Output----- Print a single integer — the number of ways you can choose the c prisoners. -----Examples----- Input 4 3 3 2 3 1 1 Output 2 Input 1 1 1 2 Output 0 Input 11 4 2 2 2 0 7 3 2 2 4 9 1 4 Output 6
