 Given two positive integers n and m, find the number of distinct subsequences of a (including the empty subsequence). 
-----Input----- The only line contains two integers n and m (1 ≤ n, m ≤ 10^6) — the number of elements in arrays and the upper bound for elements. -----Output----- Print the only integer c — the desired sum modulo 10^9 + 7. -----Examples----- Input 1 3 Output 6 Input 2 2 Output 14 Input 3 3 Output 174
