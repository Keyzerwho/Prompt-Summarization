 You are given a necklace. You need to make the number of links between every two adjacent pearls equal. 
-----Input----- The only line of input contains a string $s$ ($3 \leq |s| \leq 100$), representing the necklace, where a dash '-' represents a link and the lowercase English letter 'o' represents a pearl. -----Output----- Print "YES" if the links and pearls can be rejoined such that the number of links between adjacent pearls is equal. Otherwise print "NO". You can print each letter in any case (upper or lower). -----Examples----- Input -o-o-- Output YES Input -o--- Output YES Input -o---o- Output NO Input ooo Output YES
