 You are given $n$. You need to find the number of cyclic permutations of length $n$. Since the number may be very large, output it modulo $10^9+7$. 
-----Input----- The first and only line contains a single integer $n$ ($3 \le n \le 10^6$). -----Output----- Output a single integer $0 \leq x < 10^9+7$, the number of cyclic permutations of length $n$ modulo $10^9+7$. -----Examples----- Input 4 Output 16 Input 583291 Output 135712853 -----Note----- There are $16$ cyclic permutations for $n = 4$. $[4,2,1,3]$ is one such permutation, having a cycle of length four: $4 \rightarrow 3 \rightarrow 2 \rightarrow 1 \rightarrow 4$. Nodes $v_1$, $v_2$, $\ldots$, $v_k$ form a simple cycle if the following conditions hold:   $k \geq 3$.  $v_i \neq v_j$ for any pair of indices $i$ and $j$. ($1 \leq i < j \leq k$)  $v_i$ and $v_{i+1}$ share an edge for all $i$ ($1 \leq i < k$), and $v_1$ and $v_k$ share an edge.
