 You are given M pieces of information. Determine if there exists a set of values (x_1, x_2, ..., x_N) that is consistent with the given pieces of information. 
-----Input----- Input is given from Standard Input in the following format: N M L_1 R_1 D_1 L_2 R_2 D_2 : L_M R_M D_M -----Output----- If there exists a set of values (x_1, x_2, ..., x_N) that is consistent with all given pieces of information, print Yes; if it does not exist, print No. -----Sample Input----- 3 3 1 2 1 2 3 1 1 3 2 -----Sample Output----- Yes Some possible sets of values (x_1, x_2, x_3) are (0, 1, 2) and (101, 102, 103).
