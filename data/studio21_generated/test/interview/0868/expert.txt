 You need to print the last digit of 1378^{n}. 
-----Input----- The single line of input contains one integer n (0  ≤  n  ≤  10^9). -----Output----- Print single integer — the last digit of 1378^{n}. -----Examples----- Input 1 Output 8 Input 2 Output 4 -----Note----- In the first example, last digit of 1378^1 = 1378 is 8. In the second example, last digit of 1378^2 = 1378·1378 = 1898884 is 4.
