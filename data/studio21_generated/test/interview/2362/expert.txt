 Given a tree, calculate the maximum value of distance between two vertices. 
-----Input----- The first line contains one integer $n$ — the number of vertices $(1 \le n \le 2 \cdot 10^5)$. The second line contains $n$ integers $a_1$, $a_2$, ..., $a_n$ $(1 \le a_i \le 2 \cdot 10^5)$ — the numbers written on vertices. Then $n - 1$ lines follow, each containing two integers $x$ and $y$ $(1 \le x, y \le n, x \ne y)$ denoting an edge connecting vertex $x$ with vertex $y$. It is guaranteed that these edges form a tree. -----Output----- If there is no pair of vertices $x, y$ such that $g(x, y) > 1$, print $0$. Otherwise print the maximum value of $dist(x, y)$ among such pairs. -----Examples----- Input 3 2 3 4 1 2 2 3 Output 1 Input 3 2 3 4 1 3 2 3 Output 2 Input 3 1 1 1 1 2 2 3 Output 0
