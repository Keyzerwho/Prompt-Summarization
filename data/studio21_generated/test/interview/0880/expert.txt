-----Input----- The only line contains one integer $n$ ($1 \leq n \leq 10^6$), as described in the problem statement. -----Output----- Output a single integer — the number of subarrays of length $n$ having sum $\frac{n(n+1)}{2}$, modulo $998244353$. -----Examples----- Input 3 Output 9 Input 4 Output 56 Input 10 Output 30052700 -----Note----- In the first sample, there are $16$ subarrays of length $3$. Their sums are $6$, $6$, $7$, $6$, $7$, $5$, $6$, $6$, $8$, $6$, $7$, $5$, $6$, $6$, $7$, $6$. As $\frac{n(n+1)}{2} = 6$, the answer is $9$.
