 Given string s, check if it can be split into two non-empty subsequences such that the strings formed by these subsequences are adorable. 
-----Input----- The only line contains s (1 ≤ |s| ≤ 10^5) consisting of lowercase latin letters. -----Output----- Print «Yes» if the string can be split according to the criteria above or «No» otherwise. Each letter can be printed in arbitrary case. -----Examples----- Input ababa Output Yes Input zzcxx Output Yes Input yeee Output No -----Note----- In sample case two zzcxx can be split into subsequences zc and zxx each of which is adorable. There's no suitable partition in sample case three.
