 Given a positive integer $n$, find two integers $a,b$ such that $a-b=n$. 
-----Input----- The input contains one integer $n$ ($1 \leq n \leq 10^7$): the given integer. -----Output----- Print two composite integers $a,b$ ($2 \leq a, b \leq 10^9, a-b=n$). It can be proven, that solution always exists. If there are several possible solutions, you can print any.  -----Examples----- Input 1 Output 9 8 Input 512 Output 4608 4096
