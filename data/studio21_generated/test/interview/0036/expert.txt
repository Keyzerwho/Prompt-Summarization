 Ayrat is searching through a field. He started at point (0, 0) and is moving along the spiral. Help Ayrat determine his location after n moves. 
-----Input----- The only line of the input contains integer n (0 ≤ n ≤ 10^18) — the number of Ayrat's moves. -----Output----- Print two integers x and y — current coordinates of Ayrat coordinates. -----Examples----- Input 3 Output -2 0 Input 7 Output 3 2
