 
-----Input----- Input is given from Standard Input in the following format:   N M   a_1 b_1 c_1   a_2 b_2 c_2 :   a_M b_M c_M   -----Output----- Print the maximum possible score at the end of the game, if it is finite. If it is possible to increase the score indefinitely, print inf.   -----Sample Input----- 3 3 1 2 4 2 3 3 1 3 5 -----Sample Output----- 7 There are two ways to move the piece to vertex N=3:  - vertex 1 → vertex 2 → vertex 3 : score 4+3=7  - vertex 1 → vertex 3 : score 5 Thus, the maximum possible score at the end of the game is 7.
