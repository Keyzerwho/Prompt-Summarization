 You are given an integer b and an integer a. Return the integer representation of the number of times the phoenix will reincarnate in a timespan of b! years. 
-----Input----- The first and only line of input contains two space-separated integers a and b (0 ≤ a ≤ b ≤ 10^18). -----Output----- Output one line containing a single decimal digit — the last digit of the value that interests Koyomi. -----Examples----- Input 2 4 Output 2 Input 0 10 Output 0 Input 107 109 Output 2 -----Note----- In the first example, the last digit of $\frac{4 !}{2 !} = 12$ is 2; In the second example, the last digit of $\frac{10 !}{0 !} = 3628800$ is 0; In the third example, the last digit of $\frac{109 !}{107 !} = 11772$ is 2.
