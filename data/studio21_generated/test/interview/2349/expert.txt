 You are given $n$ rating units. You need to find a sequence of all possible rating increments. 
-----Input----- The first line contains integer number $t$ ($1 \le t \le 10$) — the number of test cases in the input. Then $t$ test cases follow. Each line contains an integer $n$ ($1 \le n \le 10^9$) — the total number of the rating units being drawn. -----Output----- Output the answers for each of $t$ test cases. Each answer should be contained in two lines. In the first line print a single integer $m$ — the number of different rating increment values that Vasya can get. In the following line print $m$ integers in ascending order — the values of possible rating increments. -----Example----- Input 4 5 11 1 3 Output 4 0 1 2 5  6 0 1 2 3 5 11  2 0 1  3 0 1 3
