 Given n, your task is to count how many right-angled triangles with side-lengths a, b and c that satisfied an inequality 1   a   b   c   n. 
-----Input----- The only line contains one integer n (1 ≤ n ≤ 10^4) as we mentioned above. -----Output----- Print a single integer — the answer to the problem. -----Examples----- Input 5 Output 1 Input 74 Output 35
