 You are given a list of days. Each day has 6 digits. Each digit is either a 1 or a 0. You are given three numbers y, n and m. Return three numbers z, n' and m' such that z = y + n' and m = m + y'. 
-----Input----- The only line contains integer y (1000 ≤ y < 100'000) — the year of the calendar. -----Output----- Print the only integer y' — the next year after y when the calendar will be the same. Note that you should find the first year after y with the same calendar. -----Examples----- Input 2016 Output 2044 Input 2000 Output 2028 Input 50501 Output 50507 -----Note----- Today is Monday, the 13th of June, 2016.
