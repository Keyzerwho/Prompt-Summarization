 You are given m cards with numbers. You want to put some cards on the table in a row to get a stair sequence. What is the maximim number of cards you can put on the table 
-----Input----- The first line contains integer m (1 ≤ m ≤ 10^5) — the number of Sereja's cards. The second line contains m integers b_{i} (1 ≤ b_{i} ≤ 5000) — the numbers on the Sereja's cards. -----Output----- In the first line print the number of cards you can put on the table. In the second line print the resulting stairs. -----Examples----- Input 5 1 2 3 4 5 Output 5 5 4 3 2 1 Input 6 1 1 2 2 3 3 Output 5 1 2 3 2 1
