-----Input----- The first line contains a single integer $t$ ($1 \leq t \leq 1000$)  — the number of test cases. The next $t$ lines describe test cases. Otherwise, print "NO" (without quotes). You can print each letter in any case (upper or lower). -----Example----- Input 7 100 3 4 189 3 4 64 2 3 63 2 3 30 27 7 10 9 1 69117 21 2 Output YES NO NO YES YES YES YES -----Note----- One possible casting sequence of the first test case is shown below: Void Absorption $\left\lfloor \frac{100}{2} \right\rfloor + 10=60$.
