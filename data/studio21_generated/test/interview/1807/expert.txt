 You are given a and b, find the number segments that  will be printed on the calculator. 
-----Input----- The only line contains two integers a, b (1 ≤ a ≤ b ≤ 10^6) — the first and the last number typed by Max. -----Output----- Print the only integer a — the total number of printed segments. -----Examples----- Input 1 3 Output 12 Input 10 15 Output 39
