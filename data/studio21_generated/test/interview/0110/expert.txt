 You are given an array $a=[a_1, a_2, \dots, a_n]$. You are allowed to perform the following operation on the array: choose any index $i$ ($1 \le i \le n$) and do $a_i := -a_i - 1$. For example, he can change array $[3, -1, -4, 1]$ to an array $[-4, -1, 3, 1]$ after applying this operation to elements with indices $i=1$ and $i=3$. 
-----Input----- The first line contains integer $n$ ($1 \leq n \leq 10^{5}$) — number of integers in the array. The second line contains $n$ integers $a_1, a_2, \ldots, a_n$ ($-10^{6} \leq a_i \leq 10^{6}$) — elements of the array -----Output----- Print $n$ numbers — elements of the array with the maximal possible product of elements which can be received using only this operation in some order from the given array. If there are multiple answers, print any of them. -----Examples----- Input 4 2 2 2 2 Output -3 -3 -3 -3  Input 1 0 Output 0  Input 3 -3 -3 2 Output -3 -3 2
