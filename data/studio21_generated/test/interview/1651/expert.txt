 
-----Input----- Input is given from Standard Input in the following format: S P -----Output----- If there is a pair of positive integers (N,M) such that N + M = S and N \times M = P, print Yes; otherwise, print No. -----Sample Input----- 3 2 -----Sample Output----- Yes  - For example, we have N+M=3 and N \times M =2 for N=1,M=2.
