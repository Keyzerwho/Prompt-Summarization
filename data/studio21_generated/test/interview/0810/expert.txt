 You are given an integer n and a number k. You need to return the number of excellent numbers with length exactly n. 
-----Input----- The first line contains three integers: a, b, n (1 ≤ a < b ≤ 9, 1 ≤ n ≤ 10^6). -----Output----- Print a single integer — the answer to the problem modulo 1000000007 (10^9 + 7). -----Examples----- Input 1 3 3 Output 1 Input 2 3 10 Output 165
