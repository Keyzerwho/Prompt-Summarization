 Given $n$ non-negative integers, find $k$-th positive number whose digital root is $x$. 
-----Input----- The first line contains a single integer $n$ ($1 \le n \le 10^3$) — the number of tasks in Petya's homework. The next $n$ lines contain two integers $k_i$ ($1 \le k_i \le 10^{12}$) and $x_i$ ($1 \le x_i \le 9$) — $i$-th Petya's task in which you need to find a $k_i$-th positive number, the digital root of which is $x_i$. -----Output----- Output $n$ lines, $i$-th line should contain a single integer — the answer to the $i$-th problem. -----Example----- Input 3 1 5 5 2 3 1 Output 5 38 19
