 You are given a string $y \in \{0,1\}^n$. Return the number of different $k$ such that there exists $x \in \{0,1\}^n$ such that $y = x \oplus \mbox{shift}^k(x)$. 
-----Input----- The first line contains an integer $n$ ($1 \leq n \leq 2 \cdot 10^5$), the length of the bitstring $y$. The second line contains the bitstring $y$. -----Output----- Output a single integer: the number of suitable values of $k$. -----Example----- Input 4 1010 Output 3 -----Note----- In the first example:  $1100\oplus \mbox{shift}^1(1100) = 1010$  $1000\oplus \mbox{shift}^2(1000) = 1010$  $0110\oplus \mbox{shift}^3(0110) = 1010$  There is no $x$ such that $x \oplus x = 1010$, hence the answer is $3$.
