 You are given a string $s$. You need to output the index of $s_1$ in $s$ and the index of $s_2$ in $s$. 
-----Input----- The first line contains a single string $s$. It is guaranteed that $2\leq |s|\leq 10000$. The next two lines contain strings $s_1$ and $s_2$ respectively. It is guaranteed that $1\leq |s_1|, |s_2|\leq 5000$ and $|s_1|+|s_2|=|s|$. All the strings consists of only lowercase English letters. -----Output----- If Anthony’s claim is possible, print “yes” (without quotes). Otherwise, print “no” (without quotes). -----Examples----- Sample Input: aabcad aba acd Sample Output: yes
