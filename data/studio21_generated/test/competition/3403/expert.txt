 You are given an input of $N$ words 
-----Input----- The first line contains an integer $N$ $(1 \leq N \leq 30000)$, the number of words in the database. Each of the following $N$ lines contains a single word from the database. The words are given in the order the algorithm compares them to a query word. All words in the database will be distinct. The following line contains an integer $Q$ $(1 \leq Q \leq 30000)$, the number of words searched for. Each of the following $Q$ lines contains a single query word. All words in the input will be strings of less than $30$ lowercase letters of the English alphabet -----Output----- Output one integer per line for each query word, the number of steps the algorithm uses when searching for the word. -----Examples----- Sample Input: 5 hobotnica robot hobi hobit robi 4 robi hobi hobit rakija Sample Output: 12 10 16 7
