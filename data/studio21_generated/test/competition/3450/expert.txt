 You are given a positive integer $n$. You need to find the last 3 digits before the trailing zeroes in $n!$. If there are fewer then 3 such digits, find all of them. 
-----Input----- The input contains one line with one integer $n$ ($1\leq n\leq 10000000$). -----Output----- Output one line with the 3 digits before trailing zeroes of $n!$. If there are fewer than 3 such digits, output all of them. -----Examples----- Sample Input: 5 Sample Output: 12
