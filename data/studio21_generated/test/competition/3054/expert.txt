 You are given a string. You need to count number of syllables in it. 
-----Input----- Input will consist of a single line of text consisting of a sequence of one or more words (as defined above) separated by single blanks. The total line length will not exceed $200$ characters. -----Output----- If the words in the input line can be divided into a haiku, then print the haiku as three lines of output.  - Each line should be left-justified.  - A single space should separate each pair of words within a line.  - Upper/lower casing from the input should be preserved.  - Non-alphabetic characters terminating each word should be preserved.  - A word cannot be split across multiple lines. If the words in the input cannot be divided into a haiku, print the line of input with no changes. -----Examples----- Sample Input 1: Blue Ridge mountain road. Leaves, glowing in autumn sun, fall in Virginia. Sample Output 1: Blue Ridge mountain road. Leaves, glowing in autumn sun, fall in Virginia. Sample Input 2: Who would know if we had too few syllables? Sample Output 2: Who would know if we had too few syllables? Sample Input 3: International contest- motivation high Programmers have fun!. Sample Output 3: International contest- motivation high Programmers have fun!.
