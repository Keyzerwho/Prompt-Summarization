 =====Problem Statement===== Given $n$ positive integers $s_1, s_2, \dots, s_n$, output the number of ways they can be chosen. 
-----Input----- The first line of the input contains the integer $n$ ($1 \leq n \leq 10^5$). The second line of the input contains $n$ distinct integers $t_1, t_2, \dots t_ n$ ($1 \leq t_ i \leq n$). -----Output----- Output the number of ways in which the values of $s_ i$ can be chosen, modulo $10^9 + 7$. -----Examples----- Sample Input 1: 2 1 2 Sample Output 1: 2 Sample Input 2: 5 3 4 5 1 2 Sample Output 2: 1
