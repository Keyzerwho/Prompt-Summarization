 You are given an array of numbers. Make sets of 3 numbers. Each should have numbers in ascending or descending order. 
-----Input----- Input will consist of single line containing an integer of $1$ to $40$ digits. The number of digits in the input is the number of wheels on the counter. Numbers may contain leading zeros. -----Output----- Print a single line of output containing one integer, the minimum number of wheel advances required to produce a palindrome. -----Examples----- Sample Input 1: 0 Sample Output 1: 0 Sample Input 2: 009990001 Sample Output 2: 3 Sample Input 3: 29998 Sample Output 3: 5 Sample Input 4: 610 Sample Output 4: 4 Sample Input 5: 981 Sample Output 5: 2
