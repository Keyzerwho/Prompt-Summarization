 Given an array of integers, remove a subarray from the array, reverse the array, and calculate the GCD of the resulting array. 
-----Input:----- - The first line contains $T$, the number of test cases. - For each test case -FIrst line contains $N$. - Last line contains $N$ numbers of the array.  -----Output:----- A single integer in a new line, maximum possible GCD.  -----Constraints----- - $1 \leq T \leq 10^2$ - $1 \leq N \leq 10^4$ - $1 \leq a[i] \leq 10^9$ Summation of N for all testcases is less than $10^6$  -----Sample Input 1:----- 1 1 2 -----Sample Output 1:----- 2
