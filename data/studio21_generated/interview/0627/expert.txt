 Given two positive integers N and K, find the number of non-increasing arrays of length K, where each element of the array is between 1 and N (both inclusive). 
-----Input:----- - Two space-seperated integers, $N$ and $K$. -----Output:----- - Output in a single line, the number of possible non-increasing arrays, modulo $10^9$ $+$ $7$. -----Constraints:----- - $1 \leq N, K \leq 2000$ -----Sample Input:----- 2 5 -----Sample Output:----- 6 -----Explanation:----- - Possible Arrays, for the "Sample Case" are as follows:  - {1, 1, 1, 1, 1} - {2, 1, 1, 1, 1} - {2, 2, 1, 1, 1}  - {2, 2, 2, 1, 1} - {2, 2, 2, 2, 1} - {2, 2, 2, 2, 2} - Hence, the answer to the "Sample Case" is $6$ ($6$ % ($10^9$ $+$ $7$)).
