 Given N and K, determine if N can be formed by the sum of K prime numbers. 
-----Input:----- - First-line will contain $T$, the number of test cases. Then the test cases follow.  - Each test case contains a single line of input, two integers $N, K$. -----Output:----- For each test case, output in a single line answer as 1 or 0. -----Constraints----- - $1 \leq T \leq 10^5$ - $2 \leq N \leq 10^5$ - $1 \leq K \leq 10^5$ -----Sample Input:----- 2 12 2 11 2 -----Sample Output:----- 1 0
