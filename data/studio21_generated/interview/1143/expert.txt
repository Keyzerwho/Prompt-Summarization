 The input given are numbers. You need to find the minimum sum of two distinct numbers x and y such that x, y > 0 and xy = N. 
-----Input:----- The first line of the input contains a single integer T denoting the number of test cases. The description of T test cases follows. Each testcase contains single integer N. -----Output:----- For each testcase, output minimum x + y such that x*y = N and x,y >0. -----Constraints :----- $1 \leq T \leq 100$ $1 < N \leq 10^{12}$ -----Sample Input:----- 1 852140 -----Sample Output:----- 1929
